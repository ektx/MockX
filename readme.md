# MockX
统一入口文档管理工具，提升前后端开发联调效率。

## 功能
- 项目接口管理，对API接口模拟，减少对后台的依赖。
- 对接口进入自动文档生成，减少文档的维护成本。
- 对项目接口进行代理，提高代码的可控性与灵活性。
